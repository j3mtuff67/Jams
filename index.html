About

Built for media work

Jam's Media is a clean portfolio for edits, graphics, thumbnails, FiveM visuals, and community media. The goal is simple: show your best work, make it look premium, and make it easy to update.

This site is made for a creator portfolio, not a messy clips page. Portfolio cards are loaded from media.json, so you can update work without digging through the HTML every single time.

Images should go inside assets/portfolio/, then you add the matching file path in the admin panel or directly inside the JSON file.

Need commissions, thumbnails, server branding, or media work? Send people to your Discord and let the portfolio do the selling instead of explaining everything manually like it’s 2016.
